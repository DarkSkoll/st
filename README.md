# ST
This is my version of *ST*

## Patches

- Xresources
- font2
- w3m
- vertical center
- scrollback

## Keybindings

- copy with selection
- paste with mousewheel
- scroll with mousewheel
- scroll with shift+{PageUp,PageDown}
- zoom in with control+q
- zoom out with control+w
- reset zoom with control+e
